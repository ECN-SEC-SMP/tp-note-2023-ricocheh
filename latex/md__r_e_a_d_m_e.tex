\hypertarget{md__r_e_a_d_m_e_autotoc_md8}{}\doxysubsection{Classe Sablier}\label{md__r_e_a_d_m_e_autotoc_md8}
Cette classe est utilisée pour la gestion du temps dans le jeu. Elle utilise la librairie $<$\+Chrono$>$. Dans ce projet, le sablier est utilisé pour décompter des temps de 1 minute. Celui-\/ci est utilisé après qu\textquotesingle{}un joueur ait annoncé le nombre de déplacement qu\textquotesingle{}il compte faire pour atteindre l\textquotesingle{}objectif. Pendant son écoulement, les autres joueurs peuvent proposer un nombre de déplacement en deçà du premier joueur si c\textquotesingle{}est possible. Quand le sablier est écoulé, le joueur avec le nombre de déplacement le moins élevé peut jouer.

Pour répondre à ce cahier des charges, nous avons créé une classe \mbox{\hyperlink{class_sablier}{Sablier}} avec une méthode de qui lance le sablier, une qui l\textquotesingle{}arrête, une qui le remet à 0 et une qui permet de savoir si le temps est écoulé ou non.\hypertarget{md__r_e_a_d_m_e_autotoc_md9}{}\doxysubsection{Classe Robot}\label{md__r_e_a_d_m_e_autotoc_md9}
Cette classe est utilisée pour créer et gérer les déplacements des robots. Dans ce projet, 4 robots sont présents. Ces robots peuvent se déplacer dans quatre directions possibles (haut, bas, gauche et droite) jusqu\textquotesingle{}à rencontrer un autre robot ou bien un mur.

Pour répondre à ce cahier des charges, nous avons créé une classe \mbox{\hyperlink{class_robot}{Robot}} qui permet d\textquotesingle{}instancier les quatre robots. Dans cette classe, il y a deux méthodes qui permettent de savoir quelle sont les positions X et Y, deux méthodes qui mettent à jour les positions que ça soit en X ou en Y tout en mettant à jour le statut d\textquotesingle{}occupation sur le plateau de jeu. Pour finir, nous avosn créé une méthode déplacement qui a pour objectif de déplacer un robot dans une des quatre directions possibles selon le choix effectué par le joueur. Pour ce faire, la fonction passe de case en case et s\textquotesingle{}assure de pouvoir les traverser en fonction des murs et des robots présents sur les cases qu\textquotesingle{}il parcoure.
